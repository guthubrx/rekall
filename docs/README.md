## Quick Links

- **Install**: `uv tool install git+https://github.com/guthubrx/rekall.git`
- **Launch TUI**: `rekall`
- **Search**: `rekall search "your query"`
- **Add entry**: `rekall add bug "title" --context-interactive`
- **Start MCP**: `rekall mcp`
